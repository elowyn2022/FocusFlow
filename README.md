# FocusFlow

[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

## Overview

FocusFlow is a sleek, minimalist productivity app designed to boost your focus by managing work and break intervals using the Pomodoro technique. Customize your sessions, track your progress, and maintain peak performance throughout your day.

Built with simplicity in mind, FocusFlow eliminates distractions and helps you build lasting productivity habits.

---

## Features

- Customizable work and break timers
- Session tracking and daily productivity stats
- Desktop notifications and sound alerts
- Dark and light mode themes
- Keyboard shortcuts for quick control
- Minimal and distraction-free UI

---

## Tech Stack

- **Frontend:** React.js
- **State Management:** Redux
- **Styling:** Tailwind CSS
- **Notifications:** Web Notifications API

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/elowyn2022/FocusFlow.git
```

2. Navigate to the project directory:

```bash
cd FocusFlow
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

Open your browser and go to `http://localhost:3000` to get started.

---

## Usage

- Use the settings panel to customize your work and break lengths.
- Click the start button to begin a focus session.
- Receive notifications when sessions end.
- Track your daily progress on the dashboard.
